# AI Smart Answer Sheet Evaluator

AI Smart Answer Sheet Evaluator is an intelligent web application that automatically evaluates student answer sheets using Artificial Intelligence and Natural Language Processing techniques.

The system analyzes answers, extracts keywords, compares them with expected answers, and generates scores instantly. It helps reduce manual correction time and improves evaluation efficiency.

## Features
- Automatic answer evaluation
- Keyword extraction
- Score generation
- Fast and accurate analysis
- User-friendly interface

## Technologies Used
### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database / Tools
- Firebase
- GitHub

## Purpose
The project is designed to simplify and automate the traditional answer sheet correction process using AI-based evaluation methods.
